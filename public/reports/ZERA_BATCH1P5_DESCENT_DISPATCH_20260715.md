## 시뮬 구성

### 강림제 (yongsinDescent) — 3벌

**구성**: 1000판 × 3프리셋 (v3 룰셋 위에서, 변인 분리)

| 번 | 룰셋 | 프리셋 | 조건 |
|---|------|--------|------|
| 1 | v3 | 목화 | ENABLE_YONGSIN_DESCENT=true |
| 2 | v3 | 금수 | — |
| 3 | v3 | 토단일 | — |

**측정 지표**:
- **클리어율 (%) — balance-v3 632c30a 대비**
  - 목화: XXX% → YYY% (변화: +/-ZZ%p)
  - 금수: XXX% → YYY%
  - 토단일: XXX% → YYY%
  - **격차**: 11.67%p → YY.Y%p (±1%p 게이트)

- **강림 슬롯 처리 분포 (횟수)**:
  - 발동: 강림 슬롯에서 용신 카드 포함 시 (×2.0 배율 적용)
  - 이월: 강림 슬롯에서 용신 미포함 → 다음 슬롯으로 연기 (1회 한정)
  - 소멸: 이월 후 여전히 미포함 → 강림 미발동 (미세 연출)

  **표기 형식**:
  ```
  목화:
    강림 슬롯: 3개/게임 (일진 기반)
    실제 발동: N회 (용신 확률 35% 기반 예상 ~1회, 실측값)
    이월 → 소멸: M회

  금수: ...
  토단일: ...
  ```

- **용신 기여 딜 비중 (%)**:
  - (강림으로 증가한 딜) / 총 콤보 딜
  - 상시 ×1.3 시절 대비: 기여도 상향/하향도 계산

  **예시**:
  ```
  목화:
    균형 사주 (용신 2종): 강림 기여 ~8% (총 딜의)
    편중 사주 (용신 1종): 강림 기여 ~5%

  금수: ...
  토단일: ...
  ```

## 봇 처리 규칙

**강림 봇** (yongsinDescent 비공개):
- 강림 시각(몇 번째 턴에 강림할지)은 게임 시작 시 사전 결정
- **봇에게는 이 정보를 공개하지 않음** (유저와 동일 조건)
- 결과: 봇은 강림을 "예측 불가 보너스"로만 체감
- 강림이 봇의 평가에 미반영 → 봇과 유저의 클리어율 차이 가능 (정상)

**comboRuleset은 v3 유지**:
- 강림제 변인 분리를 위해 레시피 신규 계층 미적용
- 봇과 유저 모두 기존 "무명 치기" 체계에서만 평가
