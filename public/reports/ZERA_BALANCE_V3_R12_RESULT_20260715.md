### 판정 근거 (3종 모두)

| 항목 | 목화 | 금수 | 토단일 |
|------|------|------|--------|

| dispatch 기준 | 10%대 붕괴 | 10%대 붕괴 | 10%대 붕괴 |
| 판정 | 의존 확정 | 의존 확정 | 의존 확정 |

- 목화: 38.50% → 6.70% (-31.80%p) — 10%대 붕괴
- 금수: 36.60% → 1.40% (-35.20%p) — 1%대 붕괴 (가장 극단적)
- 토단일: 26.30% → 9.60% (-16.70%p) — 10%대 미만 붕괴

**결론: 의존 확정** — 연환은 "좋은 판의 부산물"이 아니라 **승패의 직접 결정자**다.
연환 없이는 목화/금수 클리어율이 1~7%대로 급락. 토단일도 9.6%로 임계점 미만.

### 교차 검증 (R11 §2와 정합)

R11에서 연환 미발생 게임 클리어율이 이미 1~5%로 낮았다. 이번 어블레이션(연환 배율=0으로 강제 차단)에서도 동일 결과를 재확인 — 인과 방향 명확히 확인.

R11의 "미발생 낮은 승률"이 단순 상관(나쁜 판이라 연환도 안 뜨고 지기도 함)이었다면, 어블레이션에서는 좋은 판도 포함될 것이므로 더 높은 클리어율이 나와야 한다. 그런데도 1~10%로 낮음 → 인과 관계 확인.

## §4 이든 판정 요청

### §4-1 의존 확정에 따른 선행 과제 논의

dispatch 지시에 따라 "의존 확정 → 연환 없이 이기는 화력 선행 과제 논의"를 이든에게 요청한다.

**현황 정리**

| 항목 | 수치 |
|------|------|
| 연환 없는 클리어율(목화) | 6.70% |
| 연환 없는 클리어율(금수) | 1.40% |
| 연환 없는 클리어율(토단일) | 9.60% |
| 현행 클리어율 목표 | 25~40% |
| 격차 | -16~-35%p |
| 연환 데미지 비중(목화/금수) | 평균 47~52% |
| 연환 데미지 비중(토단일) | 평균 28% |

**연환이 주식인 이유 (데미지 구조)**

목화/금수 기준:
- 연환 발생 게임의 80%+에서 연환이 총 데미지의 30% 이상 담당
- 중앙값 비중 42~45%로 사실상 "공격의 절반"이 연환

**이든 판단 요청: 두 가지 방향**

**방향 A — "연환 없이 이기는 화력" 확보 후 희귀화**
- 융합/모으기 배율 상향 또는 새 콤보 추가로 기본 화력 보강
- 목표: 연환 없이도 클리어율 20~30% 확보
- 이후 연환 성립 조건 강화(5원소 각 2장 이상 등) + 배율 상향(잭팟화)
- 예상 작업량: R13 기본 화력 보강 → R14 연환 희귀화 + 배율 재조정

**방향 B — 현행 ×8 동결, 잭팟화 보류**
- 연환을 "희귀 사주 전용 경험"으로 재정의하되 배율 변경 없이 동결
- 게임 밸런스는 현행 유지 (연환 의존 구조가 일부 사주 유형에서 자연스러운 승리 경로)
- 잭팟화(×12/×16) 논의는 기본 화력 확보 후로 연기
