### 7-1. v3r 역생 단독 시프트 (B-6 실측)

측정: `src/test/v3Baseline.test.ts` — 순정 모드(ruleset 'v3' 기본 + 강림 OFF 기본), 1000판×3프리셋.
유일한 v3 영향 diff = 역생 ×1.2뿐 → 시프트 전량이 역생 단독 효과.

| 프리셋 | balance-v3 | v3r 순정 | 역생 시프트 | 게이트 25~40 |
|--------|-----------|---------|-----------|------|
| 목화 | 39.43% | 44.00% | +4.57%p | ✗ **상한 초과** |
| 금수 | 35.93% | 38.90% | +2.97%p | ✓ |
| 토단일 | 27.77% | 33.00% | +5.23%p | ✓ |

- **방향**: 3프리셋 전부 양(+). 역생은 순버프(적이 나를 생 → 내 딜 ×1.2)이므로 상승만 가능 — 부호 정합.
- **산술 정합**: 역생은 5원소 휠에서 공격당 약 1/5(≈20%) 빈도, ×1.2 → 평균 딜 +약 4%. 승/패 문턱 근처 비선형 증폭으로 +3~5%p 클리어율 상승은 산술 범위 내.
- **⚠ 목화 44.00% 게이트 상한(40) 초과** — balance-v3 시점 39.43(상단 근접)에서 역생 +4.57로 이탈. 이든 판정 필요: (a) 목화 프리셋 게이트 상한 재조정 / (b) 역생 배율 하향 / (c) 다른 층 난이도 상향. 예상(+0~2%p)보다 큼은 1000판 분산(±5%p)과 문턱 비선형성 합산.
