# 콤보 규칙 v4 정본 — "자유 성립 · 투입 위계 · 황금비 곡선" (이든 G3 확정)

> 지위: v4의 유일한 정의 문서. docs/SPEC_combo_v4.md로 커밋.
> 배경: 정확 레시피제(recipe) 기각 — 온보딩 벽 (족보와 달리 학습 자산 없음).
> 단 레시피 실험의 검증 자산(위계 절벽·성립 사건성·촉매/연료 개념)은 본 규칙에 승계.
> 1차 구현(74a5bf5)은 §3 황금비 곡선 누락 — 본 문서 기준 완전 구현이 정본.

## §1. 성립 (자유 — 온보딩 축)

- 융합 성립 = 정확히 **두 원소**의 조합, 장수 자유 (2~5장). 화1+토1부터 옹기가마.
- 3원소 이상 혼합 = 일반기 (바닥 명명: 홑 / 짝 / 섞어치기 — 기존 체계 불변)
- 융합명·촉매/연료 방향 = 정본 10쌍 전표 그대로 (숲·샘·광맥·옹기가마·들불 /
  벼림·개간·제방·담금질·주물). 들불(목+화)의 방향 재확인 1줄 제출 의무
  (촉=목/연=화가 맞는지 — 과거 동원소 오기 전력).

## §2. 장수 계급 (성장 축)

| 투입 장수 | 접두 | 기본 배율 |
|---|---|---|
| 2장 | 소(小) | ×1.5 |
| 3장 | (무접두) | ×3.0 |
| 4장 | (무접두) | ×4.0 |
| 5장 | 대(大) | ×5.5 |

## §3. 황금비 곡선 (숙련 축 — 1차 구현 누락분, 필수)

- **황금비 = 연료:촉매 ≈ 3:2** (연료가 촉매의 1.5배). 서사: 산물의 몸은 연료 —
  "다스려지는 재료가 넉넉해야 한다."
- 최종 배율 = §2 기본 × 비율 보정:
  - 황금비 정점 = ×1.0 (만배율)
  - 정점에서 한 계단 이탈 = ×0.85
  - 두 계단 이탈 = ×0.7 (바닥 — 절벽 금지, 그 이하 없음)
  - **2장은 보정 면제** (항상 ×1.0 — 입문 조합 벌점 금지)
- 장수별 정점: 3장 = 촉1+연2 / 4장 = 촉2+연2 (짝수 장은 균형이 정점) /
  5장 = 촉2+연3
- 옹기가마 기준 전표 (10쌍 공통 공식 — 개별 하드코딩 금지):
  화1토1=1.5 / 화1토2=3.0⭐ / 화2토1=2.55 / 화1토3=3.4 / 화2토2=4.0⭐ /
  화3토1=2.8 / 화1토4=3.85 / 화2토3=5.5⭐ / 화3토2=4.68 / 화4토1=3.85
- 상태 명명 4종 (분해식 표기): 정점="정화(正火)" 계열 / 연료 과다="미온" /
  촉매 과다="과열" / 극단="폭염" — 계열별 4자어 변주 (낳는=온도 계열,
  벼리는=날 계열 등)는 구현 제안 → 이든 검수

## §4. 승계 자산 (recipe에서 검증된 값 — v3 기준 아님)

- gather5(대모으기) = **×6.5** (T20 확정값. v3의 ×5.0 아님 — 1차 구현의 오독 지점)
- gather 2~4장: 짝 ×1.5 (일반기 취급) / 겹치기 3장 ×2.5 / 4장 ×4.0
- 오행연환 ×8 (판정 1순위) / 강림 B-3 (풀강림 ×1.8·잔광 ×1.25) / 역생 ×1.2 /
  바닥 명명 / 효과 양자택일 = 값 비례 (전 장수 개방 — 2장 효과는 자연히 소박)
- **위계 확정: 일반기(≤1.5) < 융합 곡선(1.5~5.5) < 대모으기(6.5) < 연환(8)**

## §5. 검증 게이트

- 1000판 × 3프리셋, 클리어율 25~40% + 격차 ≤15
- 동봉: 정점/비정점 발동 분포 + 봇 정점 대기 성향 (평균 대기 턴 — 정점 집착으로
  인한 턴 낭비 검출) + 봇 미러링 (봇 평가가 곡선 배율을 그대로 참조하는지)
- E2E 지문: 비율 보정 계산 코드 1줄 + "화4+토1 = ×3.85" 단위 테스트 +
  gather5 ×6.5 참조 확인
- §4 dispatch + 측정 커밋 + 소비처 전수 규칙 적용
