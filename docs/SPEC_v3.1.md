## 0-A. 최상위 구조 — 접속 의식 + 2모드 (이든 확정)

접속 동선: **① 오늘의 운세** (점수·분야별, §0-B) → **② 일일 뽑기** (운세 등급이 확률 풀
결정 — 바이블 원설계) → **③ 모드 선택: [스토리] / [사냥]**.
- "오늘의 출정" 별도 런 폐지. 운세는 특정 런이 아니라 **그날 전체의 날씨**로 작동.
- **스토리 모드**: 1막+공망왕 편 (유한, §0 서사 스코프). 보스 듀얼의 무대 (§9).
- **사냥 모드**: 난이도 사다리형 무한 등반 (변질 오행 군졸 = 레이스 모델). 일일 리텐션 엔진.
  획득 = 골드·뽑기 재화·신살/가호·카드 해금·시련 진행. **§6-2 준수: 영구 파워 성장 금지 —
  성장 = 선택지·자격·기록.** (레퍼런스: 궁수의 전설의 루프에서 영구 스탯 성장만 제거)
  현행 MVP 4층 등반 = 사냥 모드의 원형으로 편입, balance-v2 = 기준 난이도의 기준선.
  무한 사다리의 부수 효과: 사주별 유불리는 "최고 도달 난이도"의 차이로 자연 표현됨.
- **PvP = Phase 4 유지** (이든 확정: 염두만, 도입 보류).

## 0-B. 오늘의 운세 v2 — "다섯 기운의 저울" (배치 2 편입)

- 총운 점수(0~100)+등급(대길~대흉)+총평 1줄.
- 분야 5항목 점수+1줄 풀이: 재물·애정·건강·일·**전투**.
- **분야 = 게임 파라미터 1:1**: 재물=골드·보상률 / 애정=축복(갈림길) 품질 / 건강=시작 HP·
  회복 효율 / 일=뽑기 확률 풀 / 전투=기세 오름·죽음 다이얼. 수치 전부 게이트.
- 산출 = 만세력 실계산 (일진 vs 사주의 합·충·형, 오행 관계). 장식 문구 금지.
- **흉일 원칙**: 페널티가 아니라 구성 변화 — 보상 총량 유지, "액땜 보상"(흉일 한정 획득물)
  존재. 접속할 이유는 매일 있어야 한다.

## 0. 세계관-메커니즘 연결 원칙

- MVP 서사 스코프: **1막 + 공망왕 편까지.** 클리프행어("시간의 궁으로 향한다")로 종결.
- 공망왕 격파 컷 복선 1줄 필수: *"달 없는 하늘에서, 열두 쌍의 눈이 그 죽음을 지켜보았다."*
- **적 기믹 = 서사의 번역** (불가침): 고신 "홀로 됨"=가호 봉인, 명외자 "폭풍격"=기운 전환(빙의),
  공망왕 = 무속성 턴(빈 시간) 예약. 이후 적 추가 시에도 서사 근거 필수.
- **서사 위계 정정 (이든 확정)**: 빈 운명 = 공망왕 + 장군 수십 명 한정 (= 명외자의 정의).
  변질 오행 군대(수만)는 **유명(有命)자** — 사주를 갖고 태어났으나 선동된 군졸이며,
  "사주를 다룰 줄 모르는 자"이므로 레이스 모델로 싸운다 (덱 없음의 서사 근거).
  명외자 장군들은 화개가 연결한 **겁살(블랙홀)의 힘으로 타인의 사주를 흡수·사용**한다
  — 빈 자이기에 남의 운명을 빌려 가장 운명답게 싸우는 아이러니.
- 서사 전달 = 조각 방식: 카드 플레이버, 보스 이중 대사, 반전=클리어 보상, 도감 lore,
  일일 운세 풀이 = 서사 채널, 뱀 문양 복선.
- 1.5막 열두 문지기(십이지 시련 12종, 내 띠 첫 만남, D14 엔진) = 출시 후 업데이트.

## 1. 확정 규칙 전문 (balance-v2)

### 1-1. 코어 루프
- 일일 출정: 사주 입력 → (패 확인) → 가호 2종 장착 → 4층 등반 → 클리어/사망.
- 층 구성: 1층 일반 → 2층 혼성 → 3층 정예(고신) → 4층 보스(명외자 대장).
- 적 HP: **220 / 430 / 680 / 560** · maxPlays: **4 / 4 / 5 / 6** · 반격: 층별 1/1/2/4.
- 플레이어 HP 100. 손패 **8장** (전 시점 불변식, HAND_SIZE 상수 강제 — 리터럴 숫자 금지).

### 1-2. 덱 = 팔자
- 시작 덱 20장, 사주 오행 비례(안 A). **사주에 0인 오행 = 덱 0장** (최소 보장 폐지, T21-a).
- 덱은 런 스코프 영속 엔티티: 런 시작 시 1회 생성, 층 전환 재생성 금지, 보상 누적 적용.

### 1-3. 콤보 3종
- **모으기**: 같은 기운 2~5장. ×1.5/×2.5/×3.5/×5.0. 음양 공존 +20%.
- **융합**: 이종 기운 조합, 10쌍 고유 명칭·타격 속성·특성.
  - 낳는 5쌍 ×3.0 (들불火·옹기가마土·광맥金·샘水·숲木)
  - 벼리는 5쌍 ×3.5 (벼린검金·깎은화살木·일군밭土·맑은못水·담금불火)
  - ⚠️ 현행 구현 = "2종류 2~5장 허용" (Phase 1.9.3 변경). 원 스펙 "정확히 2장"과의
    판정은 배치 1 안건 (§3-5).
- **연환**: 5원소 전원 성립 시만. 배율 ×8 (상향 예정, §3-3). 출정당 제한 없음.

  quinn 검수 (입력 격리: TASKS.md+티켓 번호만, 실기 기준). 보고 첫 줄 [배포됨·지문]/[미배포].
- 측정 편향 기록: 봇은 유틸 융합·가호 운용의 하한 측정. 실전 상한은 측정치보다 높다.

## 3. 배치 1 — balance-v3 "규칙 대청소" (v3 대조 감사 후 착수)
한 번의 재기준선으로 일괄 처리:
1. **낳는 5쌍 양자택일 개편**: 공격 or 효과. 효과는 "공격 1회 대비 상황부 우위"로 재설계
   (초안: 자양 25%, 정화 전 원소+재발 면역, 채굴 3장). 벼리는 5쌍 = 현행 즉발.
2. **효과 = 투입 카드 값 비례** (낳는 공통 원칙): 응축 = 구성 매트릭스(배율) × 태운 값 총합(기본치)
   구조(B안). 저숫자 굽기 착취 차단.
3. **T21-c**: 연환 배율 ×8 → ×12~20 (측정 후 게이트) — 잭팟 완성.
4. **버리기 개편**: 1회당 최대 3장 제한 방향 (수치 게이트).
5. **T26 판정**: 융합 "정확히 2장" 복원 vs "2종류 2~5장" 유지 — 양자택일 개편과 연계 판정.
6. **T20**: 모으기 위계 (4장 ×3.5 = 벼리는 동률 해소).
7. 들불 재정의 (응축과 축 충돌 — 지속형 잔불 등).
전 항목 수치 게이트 + 봇 학습 + 3000판 재기준선.

- 카드 컨디션 비주얼: 기세 배지 폐지 → 카드 물성 (오름=팽팽·광택 / 죽음=해짐·탈색). 색맹 대응 + 롱프레스 툴팁.
- 연환 대의식 연출 (오방색 빛기둥). 생 관계 피격 흡수 연출 (수치 불변).
- 세운 층 라벨 (가상 연도, 판마다 랜덤) + **갈림길**: 층 클리어 시 일진 기반 축복 3택
  (런 누적, 풀 = 당일 일진+운세 등급. 층 보상과의 관계는 게이트).
- 유물 모바일 팝업 / 잔잔바리 드랍(해금 트리거) / 공망왕 컷 복선 1줄 (§0).

## 6. 불가침 원칙
1. 사주 개인화 = 유일한 훅. 2. 성장 축 3원칙: 덱=팔자(런) / 신살·가호=노력의 구조(영구,
   뽑기 대상=선택지, 파워 판매 금지) / 유물=인연(런). 3. 프리셋·사주별 전용 보정 금지.
4. 턴 수·공격 횟수 증가 효과 금지 (Phase 4 대칭 대비). 5. 낳는 ×3.0 / 벼리는 ×3.5 상수.
6. 적 기믹 = 서사 번역. 7. 가호 발동 스코프 = 출정당. 8. 상태 표현 = 카드 물성 (배지 금지).

## 9. Phase 4 (출시 후) — PvP 대칭 맞대결
- 상대 = 원소 덩어리가 아니라 "사주를 가진 플레이어 구조": 사주 비례 덱 + 일간 기반
  가호·용신 + 자기 손패·콤보. 일간은 정체성·주력 방향이지 약점 딱지가 아님.
- 상성 판정 = 일간끼리가 아니라 **그 턴 양측 콤보의 타격 속성 간** — 심리전의 축.
- 일간 유불리는 존재하되 (사주 게임의 정체성), 완화 3축: 비례 덱 / 노력의 축(가호·신살·
  유물) / 매칭 룰 (Phase 4 설계).
- 덱 전투 자격 원칙: "덱 = 사주를 다루는 자" (각성자·명외자 장군·신격·플레이어) /
  레이스 = 다루지 못하는 자 (군졸). 버리기 = 턴 교환 행동 전환 검토.
- 기술 검증 = 배치 보스 듀얼 (§4-B). 1.5막 열두 문지기, 2막 이후 서사 (태세→화개→태극→
  천을)는 유저 지표 검증 후 순차.
