## §1. 평민 카드 개편 (기존 덱 정리)

- 구성: 오행 5종 × 값 **2~10** (값 1 삭제 — 값 비례 체계에서 순수 딜 손실 카드 제거)
- **음양 속성 전면 삭제**: ○/● 표기 제거, "모으기 음양 조화 +20%" 규칙 동반 폐지
  (음양은 §2 왕족의 인장으로만 존속 — "모두의 배경"에서 "왕족의 정체성"으로)
- 시작 덱: 사주 기반 생성 규칙 불변, 평민만으로 구성

## §2. 왕·여왕 카드 (10종 — Balatro의 K·Q 대응)

- 오행당 **왕 1종(양간) + 여왕 1종(음간)** = 10종:
  왕 = 병화(丙火)·갑목(甲木)·무토(戊土)·경금(庚金)·임수(壬水)
  여왕 = 정화(丁火)·을목(乙木)·기토(己土)·신금(辛金)·계수(癸水)
  표기: 정통 천간명 + 직관 병기 (예: "병화 丙火 · 불의 왕") — 확정 표기는 이든 검수
- 값: **10 확정** (2026-07-18 게이트 PASS) — §6 파일럿서 value=12 과강 적발, 11 폐기(값
  하향은 격 붕괴), 10 단일로 확정.
- 효과 (각 계급 공통 규칙 1개 — 학습량 최소화):
  - **왕 (양의 인장)**: 포함된 융합의 **비율 판정 한 계단 승격하되 정점(×1.0) 도달 불가**
    — 최대 step1(×0.70)까지. "왕은 길을 넓혀주나, 완성은 그대의 손으로" (레버 b, 2026-07-18 이든 확정)
  - **여왕 (음의 인장)**: 포함된 융합의 **효과량 ×1.25** (당초 ×1.5에서 게이트 처방 하향, 1.25 하한 — 아래는 존재감 소멸)
  - "왕은 세게, 여왕은 깊게" — 이든의 양=공격/음=효과 구상의 압축 구현
- 획득: **시작 덱 미포함.** 층 보상·갈림길·뽑기(배치 2 경제)로만 유입
- 덱 소지 상한: 왕+여왕 합산 **최대 2~3장** (파일럿 결과로 확정) — 희소성은
  종수가 아니라 밀도로
- 오행 판정: 왕·여왕은 해당 오행 카드로 취급 (병화 = 화 — 융합·모으기·연환 재료 가능)

### §2 확정 기록 (2026-07-18 게이트 PASS — 커밋 051b0fc)

### 획득물 게이트 측정 표준 — 강제 A/B (제도 규칙, 영구)

> **이든 판정 2026-07-18**: 이후 태극·신살·가호 등 모든 획득물 게이트는 강제 A/B 방식 의무.
> 조건부 획득 델타 영구 금지.

- **A군**: 동일 시드, 첫 대상 획득물 등장 시 봇 3택 비교 우회 강제 획득 (`royalForceAcquire`).
- **B군**: 동일 시드, 해당 획득물 생성 배제 (기저 성능).
- **순수 델타 = clearRate(A, 등장 시드) − clearRate(B, 동일 시드)**.
- 25% 등장 롤은 판 유불리와 무상관 → 선택 편향 없음. 획득 강제 → 봇 선택 편향 제거.
- 채점: 전 프리셋 순수 델타 <15 (엄격 상한) + 최소 1개 프리셋 ≥+5 (효과 존재 증명).
  +5~12 = 권장 밴드 (초과는 노트, 15 미만이면 관용).
- 승계 적용: 태극 획득·신살 획득·가호 조커화 — 동일 A/B 프레임.

## §3. 태극 카드 (Balatro의 조커 대응)

- 위상: **덱 외부 — 장착 슬롯의 패시브.** 덱에 들어가지 않음 (와일드카드 안 폐기 —
  덱 내 만능패는 연환·융합 밀도를 높여 희소성 역행)
- 슬롯: **가호 슬롯(5칸) 공용** — 가호 = 타고난 조커(사주 계산), 태극 = 주운 조커
  (획득), 같은 슬롯 경쟁. 슬롯 신설 금지
- 효과 문법: "게임 규칙을 비트는" 패시브 — 좁은 조건 × 큰 보상 (Balatro 조커 문법).
  상세 목록은 §7 이후 별도 설계 (가호 조커화와 한 테이블)
- 서사: 영원세의 태극 — 폭군의 힘 조각을 줍는 구도 (최종막 복선 예약)

## §4. 연환 개정 — 계단형 3단 (G5 최종, 2026-07-18 이든 확정)

- **기본 연환**: 평민 5원소 각 1장, **×8** — 현행 조건 유지
- **어환(御環)**: 왕족(왕 또는 여왕) 1장 + 상이 4오행 (5장·5원소), **×12** — 희귀 계층.
  필수 아닌 열망 (왕족 미보유 유저도 기본 연환 가능)
- **천지어환(天地御環)**: 왕 + 여왕 + 상이 3오행 (5장·5원소), **×15** — 전설 계층.
  도감 항목 수준의 초희소. 전용 연출 예약
- **판정 우선순위**: 천지어환 → 어환 → 기본연환 (상위 우선)
- 빈도 억제 메커니즘: 강제 조건이 아니라 **왕족의 덱 밀도 압박** (20장 상한 내
  평민 대체 → 5원소 재료 밀도 자연 하강 — §6 파일럿이 하강 폭 측정)
- 왕 승격·여왕 증폭은 어환 배율에 중복 미적용 (fusion 경로에서만 효과)

## §6. 파일럿 시뮬 (지금 실행 가능 — 측정만, 구현 커밋 금지)

- 구성: "왕1 + 여왕1 포함 20장 덱" × 1000판 × 3프리셋
- 측정: ① 기본 연환 성립률 (현행 65% 대비 하강 폭 — 50% 미달 시 추가 레버
  재논의 조건부) ② 5장 대융합 발동률 변화 ③ 클리어율 (게이트 25~40 유지)
  ④ 왕 승격 / 여왕 증폭 발동 분포
- 규격: §4 dispatch + 측정 커밋 + 봇 미러링 (왕 승격·여왕 증폭을 봇 평가에 반영)

### §6 상태 — 파일럿 완료 · 가설 입증 (2026-07-18 이든 판정)

- **덱 밀도 억제 가설 실증**: 연환 성립률 65% → **18.6%** (-46%p). 왕족 2장(특히
  동오행 왕+여왕)이 5원소 재료 밀도를 압박 → 연환 재료 구성 난도 상승. 가설 방향 확인.
- **경계 노트 (억제 과잉)**: 18.6%는 목표 밴드(20~30) **하회** — 밀도 압박이 과도할 위험.
  본 구현 게이트에서 value 10~11 + 효과 완화로 20~30 밴드 회귀 조정.
- **클리어율 게이트 FAIL = 왕족 초안(value=12) 과강 적발** — 파일럿의 본분 완수. 전 프리셋
  40% 초과(목화 86.4/금수 86.9/토단일 79.9%). value=12 딜 기여 과도가 원인.
- 후속: §2 개정(value 10~11 + 효과 강도 옵션)을 본 구현 게이트 안건으로 이월.
